32  Motivation: Needed in order to filter the
33  table for plotting using Brian T's
34  line regresss program, when the table
35  has more than one subsample value for
36  the "pop" sections and/or "loci" list. For example,
37  if the user has subsampled a genepop file

56  the array will use a sub-datatype of np.string_
